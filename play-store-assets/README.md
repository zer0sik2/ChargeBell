# ChargeBell Play Store Assets

이 폴더에는 Google Play Console 등록용 이미지를 보관합니다.

- `chargebell_app_icon_512.png`: 앱 아이콘 512×512, 불투명 배경
- `chargebell_feature_graphic_01_1024x500.jpg`: 그래픽 이미지 1, 1024×500
- `chargebell_feature_graphic_02_1024x500.jpg`: 그래픽 이미지 2, 1024×500

권한 준비 화면용 앱 아이콘은 같은 디자인을 `app/src/main/res/drawable-nodpi/chargebell_permission_icon.png`에도 보관합니다.

스토어 원본은 앱 APK/AAB 리소스와 분리하여 관리하고, 권한 화면에 필요한 아이콘만 앱 리소스에 포함합니다.
